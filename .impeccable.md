## Design Context

### Users
- 开发者/技术写作者
- 在强对比深色界面上长时间阅读代码和文章
- 需要清晰的层次结构和阅读节奏

### Brand Personality
- **大胆、精确、有棱角**
- 技术感但不冷淡
- 编辑感但不矫情
- 视觉冲击强但不杂乱

### Aesthetic Direction
**Dark Editorial + Tech Edge** — 受 tonik.com 风格启发，大字体排版主导，高对比深色背景，少量暖调琥珀色点缀。像是技术杂志的深夜版。

### Theme
深色为主，无浅色模式。背景接近纯黑但带微暖调。

### Typography
- **Display**: Barlow Condensed (粗体、有力、几何感)
- **Body**: IBM Plex Sans (技术感、清晰、专业)
- **Code**: JetBrains Mono (已经是标配，保留)
- 标题使用 fluid clamp() 实现强烈的动态大小对比

### Color Palette
```
--bg:        #0A0A0F   (近黑，微暖)
--surface:   #13131A   (卡片/侧边背景)
--elevated:  #1C1C26   (hover态)
--accent:    #F59E0B   (琥珀色，主强调)
--accent-2:  #FBBF24   (亮琥珀，次强调)
--text:      #C9CDD3   (主文字)
--muted:     #5C6370   (次要文字)
--border:    #2A2A35   (分割线)
--code-bg:   #1A1A24   (代码背景)
```

### Layout Principles
- 侧边栏收窄，内容区为主
- 文章卡片：大标题 + 少量meta信息，不臃肿
- 大量留白，呼吸感强
- 不对称布局打破模板感

### Design Bans (Strict)
- ❌ border-left 侧边条
- ❌ gradient text
- ❌ 圆角卡片堆砌
- ❌ 居中所有内容
- ❌ AI配色（青色/蓝紫渐变）
